# ken-skills

Ken's skill authoring toolkit. Write, review, and improve Claude Code skills.

## Skills

| Skill | Invoke | Description |
|-------|--------|-------------|
| `write-a-skill` | `/ken-skills:write-a-skill` | Scaffold a new skill: gather requirements, draft SKILL.md, review with user |
| `review-skill` | `/ken-skills:review-skill` | Audit a skill against write-a-skill guidelines and produce a structured report |
| `enhance-skill` | `/ken-skills:enhance-skill` | Review a skill and apply improvements to structure, description, and content quality |
| `session-to-skill` | `/ken-skills:session-to-skill` | Capture the workflow of the current session and distill it into a reusable skill |

## Install

```sh
./skills.sh link ken-skills
```
