# Design Direction Reference

Vocabulary for Step 4 of the PRD process. Use this to help the user articulate tone and aesthetic during the design direction interview.

---

## Tone / Aesthetic Options

Present these as starting points, not rigid categories. The user should pick one or combine directions:

| Direction | Feel | Avoid |
|---|---|---|
| **Brutally minimal** | Almost nothing on the page — every element earns its place. White space is the design. | Adding decorative elements "just in case" |
| **Editorial / magazine** | Strong typographic hierarchy, confident layout, editorial photos. Feels like a well-designed publication. | Boxy card layouts, equal visual weight everywhere |
| **Luxury / refined** | Quiet confidence. Slow animations, generous spacing, muted palette with one sharp accent. Feels expensive without shouting. | Flashy gradients, crowded layouts |
| **Playful / toy-like** | Rounded shapes, bouncy motion, bright palette. Inviting and approachable. | Corporate rigidity, sharp angles |
| **Brutalist / raw** | Stark, intentionally rough. Exposed structure, unusual typography, high contrast. Makes a statement. | Polish that softens the effect |
| **Retro-futuristic** | Sci-fi meets nostalgia. Monospace fonts, scanlines, glows, terminal aesthetics. | Generic "tech startup" blues |
| **Organic / natural** | Soft textures, earthy palette, irregular shapes. Feels human and warm. | Hard grids, cold colors |
| **Industrial / utilitarian** | Dense information, functional layout, tool-like. Feels like a cockpit, not a marketing page. | Decorative elements that don't serve the data |
| **Maximalist** | Rich, layered, everything turned up. Intense palette, elaborate motion, pattern and texture. | Restraint applied where it kills the effect |
| **Art deco / geometric** | Precise symmetry, strong geometry, gold/black/cream palette. Feels curated and considered. | Randomness, organic shapes |

## Audience Mental States

Help the user think past "our users" to the actual moment of arrival:

- **Skeptical evaluator** — scanning for credibility signals, ready to leave. Needs social proof, clear value prop, no fluff.
- **Excited early adopter** — already sold on the category, wants to feel the product's personality. Needs delight and differentiation.
- **Tired professional** — has seen 100 SaaS landing pages. Needs something that doesn't look like the others.
- **First-time visitor from a referral** — has social trust but low context. Needs a clear "this is for you if..." moment.
- **Power user returning** — lives in the product daily. Needs density and efficiency, not marketing.
- **Decision-maker with budget** — needs to justify the purchase. Needs ROI signals, case studies, trust marks.

---

## Memorable Differentiator Examples

The one thing the user should feel or notice. Examples to prompt the right answer:

- "The hero typography is so big and confident it feels like a statement, not a headline"
- "The color palette is unlike anything else in this category — it's actually memorable"
- "The animation on scroll makes the product feel alive before you've even signed up"
- "It feels like a design agency built this, not a startup using a template"
- "The density of information feels intentional — like a Bloomberg terminal, but beautiful"
- "It's the quietest, most restrained page in the space — that itself is the differentiator"
